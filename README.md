# ECE 5210/6210 Course Documents (Spring 2026)

This repository contains the course materials for ECE 5210.

## Repository Contents

*   **`homework/`**: Assignments and problem sets.
*   **`old_exams/`**: Past exam papers for practice.
    *   `mt01/`: Midterm 1
    *   `mt02/`: Midterm 2
    *   `final/`: Final Exams
*   **`solutions/`**: Solutions to homework and exams.

## How to Clone

To download a copy of this repository to your local machine, run the following command in your terminal:

```bash
git clone <repository-url>
```

*Note: You can find the `<repository-url>` by clicking the green **Code** button on the GitHub repository page.*

## How to Watch for Changes

To get notified when new documents are uploaded:

1.  Navigate to the main page of the repository on GitHub.
2.  Click the **Watch** button (eye icon) located in the top-right corner of the page.
3.  Select **All Activity** to receive notifications for all updates (pushes, issues, etc.).
